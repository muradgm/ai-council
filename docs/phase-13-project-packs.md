# Phase 13 — Project Packs Layer

- `packages/project-packs/`
- project packs for TradeFrame, SignalScout, NAVO, and Swimly

```bash

pnpm project-packs:route "build a trading journal"
pnpm project-pack:status
pnpm project-pack:context tradeframe "build the MVP"
pnpm project-pack:sync
```

## Purpose

This phase connects the Council's reusable intelligence to actual products so Codex sessions are grounded in project context, not generic assumptions.
