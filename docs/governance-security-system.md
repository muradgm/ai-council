# Governance, Security, Permissions System

Phase 15 adds the guardrails needed before AI Council becomes an execution system.

## What it controls

- Permissions
- Approval gates
- Risky actions
- Prompt-injection handling
- Secrets handling
- Provider/data privacy routing
- Trading and finance safety
- Audit records
- Incident escalation

## Principle

Automation without governance is unsafe. Governance decides whether a task should proceed, require approval, be logged, be blocked, or be escalated.
