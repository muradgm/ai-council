# CLI Reference

## Core

```bash
pnpm council help
pnpm council init
pnpm council status
pnpm council plan
pnpm council doctor
pnpm council health
pnpm council map
```

## Routing

```bash
pnpm council route "build an AI trading journal feature"
pnpm council bootstrap TradeFrame "build the trading journal MVP"
pnpm council context TradeFrame "build the trading journal MVP"
```

## Projects

```bash
pnpm council project:list
pnpm council project:init TraderFrame "AI-first trading journal and analysis app"
pnpm council project:doctor TraderFrame
```

## Package-Specific Commands

```bash
pnpm skills:list
pnpm agents:list
pnpm engines:list
pnpm workflows:list
pnpm templates:list
```

## Diagnostics

```bash
pnpm validate:knowledge
pnpm council:doctor
pnpm repo:map
```
