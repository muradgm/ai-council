# Skill Authoring Standard

## Required Files
Every skill must include:

- `SKILL.md`
- `skill.json`
- `README.md`

## Required Folders
- `workflows/`
- `templates/`
- `checklists/`
- `examples/`
- `prompts/`
- `references/`

## `SKILL.md` Sections
- Purpose
- When to Use
- Inputs Required
- Operating Principles
- Workflow
- Quality Standards
- Common Failure Modes
- Output Formats
- Checklist

## Rule
Do not write bloated theory files. A skill should make an agent more useful in execution.
