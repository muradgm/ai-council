# Automation Specification

## Records

- Task record
- Backlog record
- Queue record
- Sprint plan
- GitHub issue draft
- Codex task prompt
- Release checklist
- Documentation update plan
- Automation report

## Minimum task fields

- `id`
- `project`
- `title`
- `description`
- `status`
- `priority`
- `risk`
- `createdAt`

## Required safety fields

- `risk`
- `approvalRequired`
- `blockedBy`
- `verification`
