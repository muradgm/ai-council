# Governance Spec

## Capability manifest

Each governance capability includes:

- `name`
- `title`
- `entrypoint`
- `category`
- `riskLevel`
- `approvalRequired`
- `inputs`
- `outputs`
- `keywords`

## Decisions

Valid decisions:

- `allow`
- `allow_with_logging`
- `approval_required`
- `deny`
- `escalate`

## Risk levels

- `low`
- `medium`
- `high`
- `critical`
