# Changelog

## v2.0.0 — Architecture and Knowledge Foundation

This release completes the 16-phase AI Council v2 architecture and knowledge foundation. Runtime capability is entering MVP hardening.

### Added

- Complete monorepo foundation.
- Modular `SKILL.md` skills layer.
- Senior agents layer.
- Decision engines layer.
- Workflow orchestration layer.
- Templates and deliverables layer.
- Memory and project runtime layer.
- CLI and developer experience layer.
- Evals and quality gates layer.
- Provider and tool integration layer.
- Observability and tracing layer.
- Local web console layer.
- Project packs for TradeFrame, SignalScout, NAVO/Flowday, and Swimly.
- Automation and task execution layer.
- Security, permissions, and governance layer.
- Final integration docs, release notes, command reference, repo map, and final validation.

### Operating recommendation

Stop expanding the platform foundation after v2.0.0. Harden one end-to-end runtime loop on real project work, then improve based on actual friction and output quality.
