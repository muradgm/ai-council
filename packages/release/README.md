# Release Package

The Release package contains final integration metadata, release policies, validation rules, release notes, and handoff guidance for AI Council v2.0.0.

It does not replace project release workflows. It defines the release state of the Council platform itself.

## Contents

- `release.json` — machine-readable release metadata.
- `RELEASE.md` — human-readable release summary.
- `handoff.md` — what to do after unzipping.
- `policies/foundation-freeze.md` — when not to keep expanding the platform.
- `checklists/v2-final-checklist.md` — final readiness checklist.
