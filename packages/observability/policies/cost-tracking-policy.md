# Cost Tracking Policy

## Purpose

Cost records help decide when to use local models, freemium providers, or premium providers.

## Required fields

Provider, model when known, task purpose, token estimate, currency, amount, and whether the task was worth the spend.

## Trading and finance rule

Costs must never pressure the system into using a weaker model for high-risk trading, legal, medical, privacy, security, or money-impacting decisions.
