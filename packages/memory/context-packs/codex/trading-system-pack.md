# Trading System Context Pack

Use for trading-related projects.

Mandatory safety boundaries:

- Do not promise returns.
- Separate education, analysis, simulation, and live execution.
- Require risk controls before execution logic.
- Log assumptions and uncertainty.
- Prioritize capital preservation.

Required sections:

- Instrument scope
- Data assumptions
- Strategy hypothesis
- Risk model
- Backtesting limitations
- Execution constraints
- Failure modes
