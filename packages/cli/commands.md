# CLI Command Reference

## council init

Prints the startup reading order.

## council status

Counts major knowledge objects.

## council doctor

Runs diagnostics across the repo structure, package scripts, and knowledge validation.

## council route

Routes a natural-language request to relevant agents, engines, workflows, skills, and templates.

## council bootstrap

Creates a Codex startup prompt for a project/task pair.

## council context

Creates a Codex context pack that includes orchestrator rules, project memory, and routing.

## project:init

Creates a new project scaffold and memory record.

## project:doctor

Checks whether a project has required repo and memory files.
