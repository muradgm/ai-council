# Risk Register: SignalScout

| Risk | Severity | Mitigation |
|---|---:|---|
| Scraping and outreach can cross legal, privacy, or platform-boundary lines. | Medium | Add explicit review gate and keep scope controlled. |
| Over-automation can produce generic spam instead of valuable, specific outreach. | Medium | Add explicit review gate and keep scope controlled. |
| Pipeline complexity can grow before one narrow revenue wedge is proven. | Medium | Add explicit review gate and keep scope controlled. |
| Audit scoring can be untrusted if criteria are opaque. | Medium | Add explicit review gate and keep scope controlled. |

## Review Cadence

Review risks after every architecture change, release candidate, or major workflow expansion.
