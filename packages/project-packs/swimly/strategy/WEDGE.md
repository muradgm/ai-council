# Wedge: Swimly

## Wedge Statement

A simple recommender that picks the best pool option based on purpose, district, price, and practical fit instead of overwhelming users with a long list.

## Why This Wedge Is Strong

- It is specific enough to guide product decisions.
- It gives the project a reason to exist beyond generic AI assistance.
- It can be tested with a small MVP.
- It creates a clear standard for what to build and what to reject.

## Wedge Tests

Before adding a feature, ask:

1. Does this strengthen the wedge?
2. Does this make the user's main decision easier?
3. Does this reduce friction in the core workflow?
4. Can this be tested without building a full platform?
5. Would removing this feature make the product meaningfully weaker?

If the answer is mostly no, defer the feature.
