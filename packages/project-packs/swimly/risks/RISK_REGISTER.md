# Risk Register: Swimly

| Risk | Severity | Mitigation |
|---|---:|---|
| Static data can become stale quickly. | Medium | Add explicit review gate and keep scope controlled. |
| Recommendation logic can feel arbitrary if scoring is not explained. | Medium | Add explicit review gate and keep scope controlled. |
| Too many filters can destroy the simple decision-first wedge. | Medium | Add explicit review gate and keep scope controlled. |
| The app can become a generic directory instead of a recommender. | Medium | Add explicit review gate and keep scope controlled. |

## Review Cadence

Review risks after every architecture change, release candidate, or major workflow expansion.
