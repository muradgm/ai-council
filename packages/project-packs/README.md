# Project Packs

Project Packs are the AI Council's product-specific operating files.

A skill is reusable knowledge. A senior agent is an expert role. A decision engine is a reusable reasoning system. A workflow is an execution path. A project pack connects all of those to one real product.

Phase 13 introduces complete packs for:

- TradeFrame
- SignalScout
- NAVO / Flowday
- Swimly

Use project packs when Codex or another coding agent needs project-specific context before planning, editing, or reviewing code.
