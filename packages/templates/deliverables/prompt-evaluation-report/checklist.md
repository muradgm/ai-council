# Prompt Evaluation Report Quality Checklist

Before delivery, verify:

- [ ] The template selected is appropriate for the user's request.
- [ ] Required inputs are present or assumptions are stated.
- [ ] The selected agents are relevant: chief-ai-officer, ai-engineer, research-lead.
- [ ] The selected skills are relevant: prompt-engineering, evals, agent-design.
- [ ] The decision engines are relevant: llm-prompt-evaluation-engine, ai-agent-design-engine.
- [ ] The output includes risks, constraints, and open questions.
- [ ] The output is specific enough to execute or review.
- [ ] No unsupported claim is presented as fact.
