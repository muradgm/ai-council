# Codex Task Prompt Quality Checklist

Before delivery, verify:

- [ ] The template selected is appropriate for the user's request.
- [ ] Required inputs are present or assumptions are stated.
- [ ] The selected agents are relevant: principal-software-architect, senior-ai-engineer, technical-writer.
- [ ] The selected skills are relevant: prompt-engineering, software-architecture, documentation.
- [ ] The decision engines are relevant: architecture-engine, llm-prompt-evaluation-engine.
- [ ] The output includes risks, constraints, and open questions.
- [ ] The output is specific enough to execute or review.
- [ ] No unsupported claim is presented as fact.
