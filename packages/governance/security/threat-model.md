# Threat Model

## Assets

- User files and private project context
- API keys, credentials, broker credentials
- Git history and unpublished code
- Project roadmaps and business strategy
- Provider usage and cost data
- Trading journal and financial notes

## Threats

- prompt injection through untrusted docs,
- accidental secret disclosure,
- destructive command execution,
- unsafe automation,
- cloud provider oversharing,
- trading/finance overreach,
- supply-chain dependency risk.
