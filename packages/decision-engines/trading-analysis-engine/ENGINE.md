# Trading Analysis Engine

## Purpose
Standardized reasoning engine for trading analysis engine decisions across AI Council.

## Inputs
- Objective
- Context
- Constraints
- Candidate options
- Evidence
- Risks

## Process
1. Define the decision.
2. Separate facts from assumptions.
3. Score options against criteria.
4. Identify failure modes.
5. Recommend one path.
6. Document trade-offs.

## Output
- Decision summary
- Option comparison
- Recommendation
- Risks and mitigations
- Next actions
