
# UX Research Engine Preflight Checklist

Before running this engine, confirm:

- [ ] The decision question is explicit.
- [ ] The project or business context is known.
- [ ] Constraints are written down.
- [ ] Existing repo/project files were checked if relevant.
- [ ] Required skills were loaded.
- [ ] Responsible senior agent is identified.
- [ ] Unknowns are separated from assumptions.
- [ ] High-stakes boundaries are recognized.
