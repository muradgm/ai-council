
# SEO Engine Quality Gate

A decision is acceptable only if:

- [ ] It recommends one clear path.
- [ ] It explains why rejected options were rejected.
- [ ] It includes tradeoffs.
- [ ] It includes risks and mitigations.
- [ ] It states confidence level.
- [ ] It states what evidence would change the decision.
- [ ] It names the next concrete action.
- [ ] It avoids fabricated facts or fake certainty.
