
# Legal Compliance Engine Scoring Rubric

Score each criterion from **1 to 5**.

- **1** = unacceptable, unclear, or high-risk.
- **2** = weak; major gaps.
- **3** = usable; some risk remains.
- **4** = strong; minor gaps.
- **5** = excellent; evidence-backed and robust.

| Criterion | 1 | 3 | 5 | Notes |
|---|---|---|---|---|
| Jurisdiction Clarity | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Data Sensitivity | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Contract Risk | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| User Consent | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Record Keeping | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Escalation Need | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Operational Impact | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |

## Scoring rules

- Do not hide critical flaws behind a high average.
- Mark confidence separately from score.
- Penalize unsupported claims.
- Penalize solutions that require unrealistic discipline, budget, or data.
- Prefer reversible options when evidence is thin.
