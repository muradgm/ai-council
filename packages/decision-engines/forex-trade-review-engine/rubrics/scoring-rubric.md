
# Forex Trade Review Engine Scoring Rubric

Score each criterion from **1 to 5**.

- **1** = unacceptable, unclear, or high-risk.
- **2** = weak; major gaps.
- **3** = usable; some risk remains.
- **4** = strong; minor gaps.
- **5** = excellent; evidence-backed and robust.

| Criterion | 1 | 3 | 5 | Notes |
|---|---|---|---|---|
| Market Structure | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Session Context | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Liquidity | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Invalidation | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Position Sizing | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Execution Rules | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |
| Post-Trade Review | 1 = weak / unclear | 3 = acceptable | 5 = strong / evidence-backed | |

## Scoring rules

- Do not hide critical flaws behind a high average.
- Mark confidence separately from score.
- Penalize unsupported claims.
- Penalize solutions that require unrealistic discipline, budget, or data.
- Prefer reversible options when evidence is thin.
