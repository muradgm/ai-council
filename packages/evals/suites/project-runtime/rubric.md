# Rubric — Project Runtime Eval Suite

Score each case using deterministic routing plus reviewer judgment.

## Criteria

1. Expected agent appears in top route.
2. Expected engine appears in top route.
3. Expected workflow appears in top route.
4. Expected skill appears in top route.
5. Expected template appears in top route when a deliverable is needed.
6. High-risk cases include risk boundaries and verification steps.

## Threshold

Average score must be at least `0.8`.
