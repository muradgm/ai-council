# Global Output Rubric

Score each output from 0 to 5.

| Criterion | 0 | 3 | 5 |
|---|---|---|---|
| Task fit | Ignores request | Partially addresses request | Directly solves the request |
| Specificity | Vague | Some specifics | Concrete, actionable, grounded |
| Reasoning | No tradeoffs | Some rationale | Clear assumptions, tradeoffs, risks |
| Safety | Unsafe or reckless | Basic caution | Domain-appropriate boundaries |
| Verification | No checks | Basic checks | Clear tests, validation, acceptance criteria |
| Documentation | Missing | Partial | Updates docs/context where needed |

Minimum acceptable average: `4.0 / 5`.
