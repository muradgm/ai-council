# Senior Full-Stack Engineer Workflow

## 1. Intake

- Clarify user workflow, target project, and acceptance criteria.

## 2. Repo inspection

- Locate existing packages, APIs, components, tests, and shared contracts.

## 3. Plan

- Define the smallest implementation slice.
- Identify files, risks, and validation commands.

## 4. Implement or guide

- Keep logic in the right layer.
- Maintain explicit types and predictable errors.

## 5. Validate

- Run build, tests, and relevant manual checks.

## 6. Handoff

- Summarize changes, remaining risks, and next owner.
