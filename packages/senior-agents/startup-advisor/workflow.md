# Startup Advisor Workflow

## 1. Intake

- What does the user want changed, decided, reviewed, or built?
- What project or package is affected?
- What constraints are explicit?
- What constraints are implied?

## 2. Agent fit

Confirm this role is appropriate. If not, recommend a better lead agent or a council group.

## 3. Skill loading

Load the skills listed in `skill-map.md`. Load dependencies only when they materially improve the outcome.

## 4. Work execution

Produce one or more of:

- implementation plan,
- code changes,
- audit report,
- strategy memo,
- design brief,
- checklist,
- decision record,
- next-action plan.

## 5. Review

Check:

- correctness,
- risk,
- maintainability,
- documentation,
- handoff clarity.

## 6. Handoff

Identify the next agent if another discipline should continue.
